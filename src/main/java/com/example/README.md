# Finance Tracker App

A full-stack personal finance tracking application designed to help users categorize expenses and manage budgets in real-time.

## 🚀 Features
* **Automated Expense Categorization**: Smart sorting of transactions.
* **Real-time Budgeting**: Monitor your spending as it happens.
* **Transaction Management**: Secure handling of financial data.

## 🛠️ Tech Stack
* **Backend**: Java with Spring Boot
* **Database**: SQL (using H2 for development)
* **Server**: Apache Tomcat
* **Version Control**: Git/GitHub

## 📋 How to Run
1. Clone the repository.
2. Ensure you have Java 17+ installed.
3. Run `./mvnw spring-boot:run` in your terminal.
